# 📝 Blog Platform (Django)

A full-stack Blog Platform built using Django where users can create, read, update, and delete blog posts. This project demonstrates backend development, authentication, and database management using Django.

---

## 🚀 Features

- User Authentication (Login / Register / Logout)
- Create, Edit, Delete Blog Posts
- View All Blogs
- Individual Blog Detail Page
- Admin Panel for Management
- Responsive UI using Django Templates

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap (Django Templates)
- Database: SQLite
- Version Control: Git & GitHub
